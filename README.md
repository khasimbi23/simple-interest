🧮 Simple Interest Calculator
A beginner-friendly Simple Interest Calculator built with Python. This project demonstrates core programming concepts like user input, mathematical operations, and clean code structure.

📌 What is Simple Interest?
Simple Interest is calculated using the formula:
SI = (P × R × T) / 100
Where:

P = Principal Amount
R = Rate of Interest (per annum)
T = Time (in years)


🚀 Features

✅ Takes user input for Principal, Rate, and Time
✅ Calculates Simple Interest instantly
✅ Displays the Total Amount (Principal + Interest)
✅ Clean and easy-to-understand code
✅ Perfect for beginners learning Python


🛠️ Technologies Used

Language: Python 3
Concepts: User Input, Arithmetic Operations, Variables, Functions
